# ДОМОСТРОЙ — Веб-приложение для строительной компании

## Описание проекта

Веб-приложение для строительной компании «Домострой». Сайт позволяет клиентам ознакомиться с услугами, рассчитать стоимость строительства через калькулятор сметы и оставить заявку на обратную связь. Для сотрудников компании реализована административная панель с возможностью управления услугами и просмотра заявок.

## Функциональные возможности

### Для клиентов:
- Просмотр каталога строительных услуг
- Поиск и фильтрация услуг по категориям
- Расчёт стоимости в калькуляторе сметы
- Отправка заявки через форму обратной связи
- Просмотр портфолио реализованных проектов

### Для администратора:
- Вход в защищённую административную панель
- Управление услугами (добавление, редактирование, удаление)
- Просмотр всех поступивших заявок
- Изменение статуса заявок (Новая / В работе / Выполнена)
